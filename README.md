QuizForge is a smart web-based application developed using React that automatically generates quizzes
from uploaded documents. The system analyzes the content of the uploaded file and creates customized
quiz questions based on user preferences such as difficulty level, number of questions, and selected topic.
The project is designed to help students, teachers, and self-learners save time while preparing assessments
and improving learning efficiency. Instead of manually creating quizzes, users can upload study material in
document form and instantly receive a structured quiz.
QuizForge uses document analysis and intelligent content processing to identify important concepts from
the uploaded material. The generated quiz can include multiple levels of complexity, making it useful for
beginners as well as advanced learners.
The application provides a clean and interactive user interface built with React, ensuring smooth user
experience, fast rendering, and responsive design
